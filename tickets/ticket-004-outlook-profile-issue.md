# Ticket 004: Outlook Profile Issue

## Issue
Outlook would not open for the user.

## Symptoms
- App froze at launch
- Error loading profile

## Troubleshooting Steps
1. Opened Outlook in safe mode
2. Disabled add-ins
3. Created a new mail profile
4. Reattached mailbox
5. Tested send/receive

## Resolution
New profile resolved the issue.

## Root Cause
Corrupted Outlook profile.
