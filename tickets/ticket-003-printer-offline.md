# Ticket 003: Printer Offline

## Issue
User was unable to print to office printer.

## Symptoms
- Printer displayed as offline
- Print jobs stuck in queue

## Troubleshooting Steps
1. Confirmed printer power and network connection
2. Restarted printer
3. Cleared print queue
4. Restarted Print Spooler service
5. Re-added printer on workstation
6. Tested print successfully

## Resolution
Printer came back online and test page printed successfully.

## Root Cause
Print queue and spooler service issue.
